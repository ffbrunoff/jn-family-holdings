# J.N Family Holdings LLC — Institutional Website

Static institutional website for J.N Family Holdings LLC, a technology company specializing in software and mobile application development.

## Stack
- Plain HTML, CSS, and a touch of vanilla JS
- No build step — deploys directly to Vercel as a static site

## Local preview
Open `index.html` in a browser, or run any static server:

```bash
npx serve .
```

## Deploy
Configured for Vercel static hosting via `vercel.json`.
